Create branch
$ git branch <branchname>

Change to branch
$git checkout <branchname>

Differences
$ git diff

	options
	--staged // changes between stage and last commit

Reset local changes
$ git checkout
	
	options
	<filename> // reset local changes on a file

Reset the stage
$ git reset HEAD
	
	options
	<filename> // unstage file

Remove commit
$ git reset <commit>
	
	options
	--hard // remove changes

Amend last commit
$ git commit --amend


Log and alias
$ git log
 
	//options
	--graph
	--oneline
	-p
	-1
	--decorate

Create alias
$ git config --global alias.lg 'log --graph --oneline --decorate -all'

New Git Commands
$ git add 
$ git commit -m "<message>"
$ git status

Setup Git
$ git config --global user.name "<name>"
$ git config --global user.email <email>

Initialize git repo
$ git init
